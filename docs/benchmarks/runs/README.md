# Benchmark Runs

## Purpose

这个目录存放 benchmark 的实际运行记录。

目标不是只保留一个总分文件，而是为每个场景留下：

- 输入
- 比较条件
- 观察到的工件
- rubric 评分
- pairwise 判断
- 后续动作
